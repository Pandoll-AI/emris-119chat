# Pre-KTAS → 응급의료센터 등급 추천 전략 v1.0

*생성일: 2026-04-24*

## 1. 연구 목적

이전 phase에서 "Pre-KTAS → Y코드 1개 확정" 프레이밍은 임상적으로 unsafe함이 확인됨 (진단 확정은 구급대원 현장에서 불가). 본 phase는 프레이밍을 **"Pre-KTAS → 응급의료센터 등급(권역/지역센터/지역기관) 추천"**으로 전환한다. 진단 확정 없이도 **필요한 자원 등급만 판정**하면 안전한 현장-병원 연결이 가능하다. 핵심 전략은 **"권역 세이브"** — 지역센터로 처리 가능한 경우는 지역센터를 우선 추천해 권역 자원을 진짜 필요한 환자에게 남긴다.

## 2. 입력 / 출력

| 구분 | 파일 | 내용 |
|---|---|---|
| 입력 | `research/prektas-to-y-mapping.json` | v0.1 Pre-KTAS → Y코드 후보 매핑 |
| 입력 | `data/y-code-to-center-tier.json` | 27 Y코드 tier 분류 1차안 (임상 리뷰 전) |
| 입력 | `data/prektas-codebook.json` | grade 참조 |
| 출력 | `research/prektas-tier-recommendation.json` | per-code tier 추천 + 요약 |

## 3. 규칙

```
Pre-KTAS 엔트리 → tier 추천
├─ Y코드 후보 있음
│  ├─ 후보 중 하나라도 지역센터 가능 → primary=지역센터, acceptable=[지역센터, 권역]
│  │                                 regional_save_applied=true
│  └─ 모든 후보가 권역 전용           → primary=권역, acceptable=[권역]
├─ Y코드 후보 없음 + grade 1–2        → primary=지역센터, acceptable=[지역센터, 권역]
│                                       regional_save_applied=true
├─ Y코드 후보 없음 + grade 3          → primary=지역기관, acceptable=[지역기관, 지역센터]
└─ Y코드 후보 없음 + grade 4–5        → primary=지역기관, acceptable=[지역기관, 지역센터]
```

### Y코드 tier 분류 1차안

| 분류 | Y코드 | 비고 |
|---|---|---|
| **권역 전용 (12)** | Y0010 심근경색, Y0020 뇌경색, Y0031 거미막하, Y0032 뇌내출혈, Y0041 흉부대동맥, Y0042 복부대동맥, Y0100 저체중아, Y0120 중증화상, Y0131 수지접합, Y0132 사지접합, Y0171/0172 혈관중재 | 24/7 권역급 인프라 필요 (PCI·stroke team·신경외과·NICU·화상센터·수부외과·IR) |
| **지역센터 가능 (15)** | Y0051/0052 담낭담도, Y0060 복부응급수술, Y0070 소아장중첩, Y0081/0082 위장관 내시경, Y0091/0092 기관지 내시경, Y0111/0112/0113 산부인과, Y0141/0142 투석, Y0150 정신과, Y0160 안과 | 지역센터급에 해당 과 당직 있으면 커버. **이 그룹이 곧 권역 세이브 대상.** |

## 4. 결과

### 4-1. 전체 tier 분포

| Primary tier | 건수 | 비율 |
|---|---:|---:|
| 권역응급의료센터 | 398 | 8.5% |
| 지역응급의료센터 | 2,341 | 49.9% |
| 지역응급의료기관 | 1,950 | 41.6% |
| 합계 | 4,689 | 100% |

**해석**: 4,689개 Pre-KTAS 코드 중 권역이 1차 선택인 경우는 **8.5%에 불과**. 나머지 91.5%는 지역센터 또는 지역기관에서 처리 가능. 기존에 모든 중증을 권역으로 보내는 관행과 비교하면 상당한 자원 효율화.

### 4-2. 권역 세이브율

**권역 세이브 적용 가능(save-eligible)**한 경우 = Y코드 후보가 있거나 grade 1–2인 케이스.
- save-eligible: 2,739건
- save-applied: 2,341건
- **세이브율: 85.5%**

즉, **권역으로 보낼 수도 있었던 2,739건 중 85.5%를 지역센터로 돌려**, 권역 자원을 398건의 진짜 권역 필수 케이스에 집중시킬 수 있다.

### 4-3. 추천 source 분포

| Source | 건수 | 비율 |
|---|---:|---:|
| grade 1–2 unmapped → 지역센터 | 1,850 | 39.5% |
| grade 3 unmapped → 지역기관 | 1,218 | 26.0% |
| grade 4 unmapped → 지역기관 | 516 | 11.0% |
| Y후보(지역센터 가능) → 지역센터 | 491 | 10.5% |
| Y후보(권역 전용) → 권역 | 398 | 8.5% |
| grade 5 unmapped → 지역기관 | 216 | 4.6% |

**가장 큰 그룹이 "grade 1–2 unmapped"인 점**이 중요. 내과적 중증(물질오용 쇼크, 패혈증, 대사 위기 등)은 27 Y코드 외지만 grade는 1–2. 이 그룹에 지역센터 추천이 붙으면서 권역 세이브가 대량 발생.

## 5. 주요 발견

### 5-1. 권역 자원 5.5배 절감 가능성

기존에 "grade 1–2 중증은 모두 권역으로" 관행을 적용하면 약 2,352건(grade 1+2)이 권역 대상. 본 전략에서는 398건만. **약 5.9배 축소**. 실제 운영에서도 85.5% 세이브가 가능하다는 것은, 권역센터가 수용 가능한 병상이 제한적인 현실에서 매우 유의미하다.

### 5-2. 신경계의 "권역 필수" 비율이 높음

신경계 345건 중 188건(54.5%)이 권역 필수 — 뇌경색·뇌출혈 감별이 현장 불가능하기에 후보 3개가 모두 권역이 되어버림. **5-3 논의와 연결**: 신경학적 증상군을 2–3개 구분(의식변화만, FAST만, 두부외상만)해서 tier를 내릴 수 있는지 임상 리뷰 가치 있음.

### 5-3. Y코드 후보 mixed 케이스는 본 규칙에서 자동 세이브

"하나라도 지역센터 가능" 규칙 덕분에 Y후보가 뒤섞인 경우(예: Y0111(분만, 지역센터) + Y0112(산과수술, 지역센터))는 자연스럽게 지역센터로 갔지만, 현재 v0.1에서는 모든 산부인과 Y코드가 지역센터 가능이라 mixed 케이스가 발생하지 않음. 향후 Y코드 수가 늘거나 tier가 더 정교해지면 이 규칙이 더 유용해질 것.

### 5-4. 정신건강·눈 카테고리의 100% 지역센터 분류

Phase 3 v0.1에서 지적된 "over-inclusion"(눈·정신건강 전체를 Y0160·Y0150 후보로 묶음)이 tier 맥락에서는 **덜 위험**하다. over-inclusion해도 모두 지역센터로 가므로 권역 자원에 영향 없음. 다만 **지역센터 입장에서는 여전히 over-referral 부담**이니 v0.2에서 pre-filter는 유효.

## 6. 한계 및 후속

### 6-1. 임상 리뷰 전 제한적 신뢰

- Y코드 tier 1차안(특히 Y0131/0132 수지사지접합, Y0120 중증화상)은 응급의학 전문의·심사기준 대조 필요. 수도권/비수도권에 따라 커버 가능 병원이 달라질 수 있음.
- v0.1 rule gap이 여전히 전파됨: 예) Y0042(복부대동맥) 2건만 — 실측에서는 더 많아야 정상.

### 6-2. "여유시" 해석 미정

`acceptable_tiers: [local_center, regional]` 같은 "지역센터 기본, 여유시 권역" 규칙에서 "여유시"를 무엇으로 판정할지 현재 데이터 없음. 다음 phase에서 EMRIS 실시간 병상 데이터(`emris-data/devdocs/hospitals.json`, beds.json)를 병합해야 구체화 가능.

### 6-3. 지역 격차

권역/지역센터/지역기관 분류는 행정 등급이지, 실제 capability를 담지 않음. 예) 어느 지역센터는 산부인과 없음, 어느 지역기관은 야간 의사 부재. Phase 5에서 병원별 capability 매트릭스와 결합 시 이 격차 반영 가능.

### 6-4. 이항 정확도 미측정

현재 추천의 **false negative rate**(중증인데 지역기관으로 보낸 비율)는 측정 전. Phase 3 §5-1 권고대로 `source-prektas.csv` 225k 실측으로 검증 필수.
