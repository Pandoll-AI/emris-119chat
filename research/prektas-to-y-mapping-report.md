# Pre-KTAS → EMRIS 중증응급질환(27 Y코드) 매핑 연구 v0.1

*생성일: 2026-04-24*

## 1. 연구 목적

구급대원이 현장에서 내린 Pre-KTAS 5자 코드(4,689종)를 **EMRIS 종합상황판의 27개 중증응급질환 Y코드** 중 하나로(또는 "해당 없음"으로) **0–3개 추가 질문** 내에 결정 가능한지 규명한다. 본 연구는 이 매핑이 가능한 범위, 필요한 질문의 종류와 깊이, 그리고 두 체계 사이의 체계적 비대응 영역을 정량·정성으로 측정한다.

## 2. 데이터

| 자원 | 규모 | 출처 |
|---|---|---|
| Pre-KTAS 코드북 | 4,689 entries · 17 level2 · 2,241 성인 + 2,448 소아 | `data/prektas-codebook.json` (v2.0.0) ← `data/raw/Pre-KTAS_codebook.csv` |
| EMRIS 중증응급질환 | 27 Y코드 | `data/emris-severe-emergency-diseases.json` ← `emris-data/devdocs/disease_codes.json` |

Pre-KTAS와 KTAS 사이의 `KTAS_PreKTAS_mapping.csv`는 **본 매핑에 사용하지 않는다**. KTAS 코드북을 Pre-KTAS 이름으로 오인했던 이전 오염 사고의 재발을 차단하기 위함.

## 3. 방법

### 3-1. 규칙 기반 v0.1 baseline

`scripts/research/build-prektas-to-y-mapping.mjs`는 각 Pre-KTAS 엔트리에 대해 순차 규칙(12개)을 적용해 후보 Y코드 집합과 필요한 질문 목록을 산출한다. 규칙은 `level2.name`으로 도메인을 우선 결정하고, `level2/3/4.name`의 키워드로 Y코드 후보를 좁힌다.

### 3-2. 질문 카탈로그 (13개)

후속 질문은 enumerable 선택지 형태로 사전 정의되어 있고 각각 고유 ID를 가진다. 주요 축:

| ID | 목적 |
|---|---|
| `onset_time_stroke` | 뇌경색 재관류 적응 (Y0020) |
| `trauma_or_spontaneous` | 외상성 vs 자발성 뇌출혈 |
| `chest_pain_character` / `aortic_location` | Y0010 심근경색 vs Y0041 흉부 vs Y0042 복부 대동맥 감별 |
| `pregnancy_status` | Y0111/Y0112/Y0113 산부인과 분기 |
| `bleeding_site` / `foreign_body_site` | 위장관/기도 분기 |
| `burn_severity` | Y0120 중증화상 적응 |
| `dialysis_indication` | Y0141 HD vs Y0142 CRRT |
| `replantation_part` | Y0131 수지 vs Y0132 사지 |
| `psychiatric_risk` | Y0150 응급입원 적응 |
| `eye_emergency_kind` | Y0160 안과 응급 적응 |

**주목할 점**: Pre-KTAS 코드의 첫 문자(C=성인, D=소아)가 이미 연령군을 담고 있어, Y0081 vs Y0082(성인·영유아 위장관 내시경), Y0091 vs Y0092(성인·영유아 기관지 내시경) 같은 연령 분기는 **질문 없이 결정 가능**하다. 이는 Pre-KTAS 코드 자체의 구조적 정보 가치다.

## 5. 주요 발견

### 5-1. 관찰: Pre-KTAS 중증의 대부분은 EMRIS 27 Y코드 외에 있다

Grade 1 소생 환자의 80.6%가 unmapped인 것은 rule의 불완전함이 아니라 두 체계의 목적 차이에서 비롯된다. Pre-KTAS는 **삼각분류(triage) 도구**로 환자 전반의 긴급도를 측정하는 반면, EMRIS 27 Y코드는 **수술·시술 중심 자원 디스패치** 대상을 정의한다. 따라서 다음 상황은 Pre-KTAS 상 최중증이지만 27 Y코드 외이다:

- 물질오용·중독으로 인한 쇼크·무의식·패혈증 (내과적 지지요법 대상)
- 감염성 쇼크, 당뇨병성 케톤산증 같은 내과적 위기
- 단순 중증 외상 골절(수술 적응이 단계적)
- 발작, 정신과 이외의 의식변화(뇌질환 제외)

**시사점**: 119-EMRIS 매핑의 성공 지표를 "커버리지 극대화"로 잡는 건 잘못됐다. 올바른 지표는 **"정답인 것을 정답으로, 정답 아닌 것을 정답 아닌 것으로 분류하는 이항 정확도"**이다.

### 5-2. 연령 분기는 질문 없이 해결된다

Pre-KTAS 코드 첫 문자(C/D)가 성인·소아를 이미 확정하므로, Y0081/Y0082·Y0091/Y0092·Y0171/Y0172 같은 연령 분기 쌍은 **추가 질문 0개**로 결정된다. Pre-KTAS 코드 자체가 구급대원이 이미 취한 의사결정을 담고 있다는 점의 실용적 가치다.

### 5-3. 카테고리 수준 포함은 over-inclusion 위험

현재 v0.1 규칙에서 `level2=눈` 전체를 Y0160 후보로, `level2=정신건강` 전체를 Y0150 후보로 묶고 1 질문(`eye_emergency_kind` / `psychiatric_risk`)으로 걸러내는 방식은 **over-inclusion**이다. 경증 결막염이나 단순 수면장애 상담까지 후보에 들어간다. 이는 질문 1개로 걸러지지만, 연구 정확성을 위해서는 규칙에서 **명시적 증상 키워드 기반 pre-filter**가 들어가는 게 옳다.

### 5-4. 확장이 가장 시급한 rule gap

| 카테고리 | 개선 방향 | 예상 coverage 증가 |
|---|---|---:|
| 임신/여성생식계 | 20주 이상 임신 + 복통 → Y0112 후보로 기본 포함 | ≈ 150건 |
| 소화기계 | 담낭·담도 keyword 외 급성 복막염·복강 내 출혈까지 Y0060 확대 | ≈ 80건 |
| 심혈관계 | Y0042(복부대동맥) rule이 level4 "복부 찢어지는 통증" 등을 잡지 못함 | ≈ 30건 |
| 몸통외상 | 두부외상·대량출혈·흉복부 관통상을 Y0032/Y0060/Y0112 등에 연결 | ≈ 100건 |
| 비뇨기계/남성생식계 | 고환 경색·대량 혈뇨 등 일부를 Y0060 후보에 포함 | ≈ 50건 |

이 다섯 개선만으로 coverage 약 **+10%p**(19% → 29%) 예상.

## 6. 한계

- **v0.1 rule set은 임상 검증 전**. 본 연구는 알고리즘 baseline이며, 각 매핑이 실제 119-병원 전달 의사결정에 적절한지는 응급의학 전문의 검토가 필요하다.
- **과학적 증빙이 아닌 휴리스틱**. 키워드 기반 rule은 `'가슴통증'` 같은 표현이 Pre-KTAS 코드북에 정확히 나타나는지에 의존한다. 일부 rule은 실제 Pre-KTAS 4단계 텍스트와 안 맞을 수 있다.
- **질문 트리 대신 질문 리스트**. 현재 `questions: ['q1','q2']`는 순차 트리가 아니라 "둘 다 물어라"에 해당. 실제 임상 흐름(한 답에 따라 다른 질문으로 이동)을 담으려면 tree 구조로 확장 필요.
- **`source-prektas.csv`(225k 실측 방문)** 로 validation하지 않았다. 다음 phase 과제.

## 7. 후속 연구 방향

| 우선순위 | 과제 | phase |
|---|---|---|
| 1 | §5-3 카테고리 수준 포함을 level3/4 키워드 기반 pre-filter로 교체 | P4 |
| 2 | §5-4의 다섯 rule gap을 채운 v0.2 매핑 | P4 |

## 8. 재현

```bash
npm run codebook:rebuild              # Pre-KTAS 코드북 정본 재생성·검증
npm run research:prektas-to-y-mapping # v0.1 매핑 산출
```

출력:
- `research/prektas-to-y-mapping.json` — per-code 매핑 + 질문 카탈로그 + 요약 통계
- 본 문서 — 서술형 해석
